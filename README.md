<h1 align="center">
    <br> Aramuni Wallet
</h1>

Bem-vindo ao repositório do AramuniWallet! Este projeto foi desenvolvido como parte da disciplina de Projeto de Software - 2024, e tem como objetivo criar uma aplicação web que simplifica o controle financeiro pessoal e profissional.

O sistema oferece recursos para gerenciar receitas e despesas, incluindo o registro de transações, exibição de relatórios e gráficos para análise. Utilizamos uma abordagem orientada a objetos e as melhores práticas de engenharia de software.
